# Activating NumLock On Startup (CachyOS - KDE)

### System Settings -> Input Devices -> Keyboard
### "On Login" -> Turn On

### Find /etc/sddm.conf.d/kde_settings.conf
### Add the below line:
```
[General]
Numlock=on
```
### Save and reboot
